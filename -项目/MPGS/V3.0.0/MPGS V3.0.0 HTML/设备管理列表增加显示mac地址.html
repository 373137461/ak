设备管理-探测器 业务说明：

1、去掉原管理器模块，仅保留探测器模块；

2、在探测器模块增加“ID”和“识别区号”两个字段；

（1）ID：显示视频、红外、地磁的mac地址和超声波的管理器探测器序号

（2）识别区号：显示视频的识别区号，其他设备默认“无”

设备管理-指示灯 业务说明：

1、在指示灯模块增加“ID”字段

2、显示指示灯的ID（PVD-0/红外管理器探测器序号）

设备管理-屏管理 业务说明：

1、在屏管理模块增加“组地址”和“屏地址”字段

设备管理-蓝牙 业务说明：

1、将原来的“控制器”字段改为“蓝牙”；

2、增加“ID”字段，显示蓝牙ID；

3、将原“蓝牙设备ID”字段改为“微信ID”

设备管理-19款PVD 业务说明：

1、将原“蓝牙设备ID”字段改为“微信ID”

2、增加“ID”字段，显示19款PVDID；
