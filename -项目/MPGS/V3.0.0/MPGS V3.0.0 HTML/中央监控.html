1、中央监控-元素显示树改为全部元素-在/离线设备（车位、屏、蓝牙）、路径图标（查询机、扶手梯、楼梯、升降电梯）、标识（卫生间、出口、入口、车流向）、其他（地图背景、文字标识、区域（引导区域、统计区域、室外区域、车库区域））。区域默认隐藏，其他元素均默认显示。
