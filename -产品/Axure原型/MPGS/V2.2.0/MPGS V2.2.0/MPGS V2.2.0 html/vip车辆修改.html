下载模板：批量开通VIP模板

业务说明：


1、在车辆管理中新增“VIP车辆”模块，表格为接收VEMS传来的筛选的带车位的VIP信息和MPGS系统开通的VIP记录，

筛选条件为车位号码、车主姓名、车牌号码和VIP状态，

VIP状态包含：全部、已生效、待生效、生效中、已过期、已取消

基本操作为开通VIP、批量开通VIP和取消。


2、开通VIP模块，所有信息为必填项

2.1车位号码，输入数字，支持模糊搜索，自动跳出含有相应字段的结果，点击选择后显示在输入框。支持添加多个车牌，填写完成后点击添加，已添加的车位显示在输入框下方，可接着添加下一个。

2.2车主姓名和车主电话为手动输入，车主电话须进行合法性校正

2.3支持添加多个车牌，填写完成后点击添加，已添加的车牌显示在输入框下方，可接着在输入框添加下一个，车牌号码须进行合法性校正

2.4VIP有效期，点击输入框弹出时间选择弹窗，时间精确到秒，支持多个时间段

设置后输入框显示格式：

完整时间段：例：2017/08/01 00:00:00-2017/09/30 23:59:59  YY/MM/DD AA:BB:CC-YY/MM/DD AA:BB:CC格式

分时段：例：2017/08/01-2017/09/30  18:00:00-24:00:00，表示2017年8月1日至9月30日每天的18点-24点为VIP时间段

多个时间段在表格中以逗号分隔开进行显示

2.5最近操作时间：YY-MM-DD AA:BB:CC格式，精确到秒

2.6操作-取消：仅可对已生效、待生效和生效中的VIP进行取消操作，已过期和已取消的VIP状态的操作列默认为“-”


3、批量开通VIP模块，勾选已有记录可进行批量开通VIP操作
